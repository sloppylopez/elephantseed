# Docker Node

## Getting Started
Make sure you have jspm installed before proceeding `npm install -g jspm`
Install `docker-node` globally:

```bash
$ npm install docker-node
```
### Usage

* Create a new folder for your project:

```bash
$ mkdir my-docker-node
```
